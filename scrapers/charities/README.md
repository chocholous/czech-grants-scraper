# Charitativní scrapery

Tento modul je připraven pro implementaci scraperů charitativních organizací.

## Plánované zdroje

- **Veřejný rejstřík** (justice.cz) - spolky, nadace, nadační fondy
- **ARES** (ares.gov.cz) - ekonomické subjekty
- **Neziskovky.cz** - databáze NNO
- **CEDR** (cedr.mfcr.cz) - příjemci dotací
