# Nadační scrapery

Tento modul je připraven pro implementaci scraperů nadací a nadačních fondů.

## Plánované zdroje

- **Nadace OSF** (osf.cz) - Open Society Fund Praha
- **Nadace Via** (nadacevia.cz) - komunitní rozvoj
- **Nadace Partnerství** (nadacepartnerstvi.cz) - ekologie
- **Kellner Family Foundation** - vzdělávání
